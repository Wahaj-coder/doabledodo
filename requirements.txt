# ── Install steps ────────────────────────────────────────────
# 1. pip install -r requirements.txt
#
# 2. Install Ollama:
#    Linux/Mac: curl -fsSL https://ollama.com/install.sh | sh
#    Windows:   https://ollama.com/download
#
# 3. Pull embedding model:
#    ollama pull nomic-embed-text        (768-dim, smaller, faster)
#    ollama pull qwen3:embedding          (1024-dim, better quality)
#
# 4. Start Ollama server:
#    ollama serve
#
# 5. Qdrant (choose one):
#    Option A — local file mode (no setup, just set QDRANT_MODE=local in .env)
#    Option B — Docker:
#        docker run -p 6333:6333 -v $(pwd)/qdrant_storage:/qdrant/storage qdrant/qdrant
#    Option C — Hetzner server: same Docker command, set QDRANT_MODE=server, QDRANT_URL in .env
#
# 6. Test pipeline:
#    python embedder.py         (test Ollama connection)
#    python qdrant_store.py     (test Qdrant connection)
#    python ingestor.py --repo https://github.com/pallets/click
#
# 7. Start servers:
#    uvicorn webhook_server:app --host 0.0.0.0 --port 8000
#    uvicorn search_api:app     --host 0.0.0.0 --port 8001
#
# 8. For ngrok (testing webhooks locally):
#    ngrok http 8000
#    Copy https URL → GitHub repo settings → Webhooks
